title: Installing Imager::File::GIF and Imager::File::JPEG with homebrew date: 2010-12-27 rownav: 1

More for my future reference than anything else :)

brew install giflib
cpanm --look Imager::File::GIF
    perl Makefile.PL --libpath=~/.brew/lib --incpath=~/.brew/include
    make test install

brew install libjpeg
cpanm --look Imager::File::JPEG
    perl Makefile.PL --libpath=~/.brew/lib --incpath=~/.brew/include
    make test install
